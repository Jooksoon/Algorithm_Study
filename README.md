### :pushpin: Pull Request 규칙
***
  * 온라인 미팅 전에 PR 진행
  * PR 제목 : 이름 / 주차 / 문제 수
  * <code> 홍길동 / 8월 1주차 / 4문제 </code>
  * PR 내용(자유롭게 작성)
  * 참고
  ```
  [문제번호] 문제이름
    - 어려웠던 점, 관련 내용, 참고 자료 등 회고 내용
  ```
  
### :pushpin: 구조
***
  * 이름 / 주차 / 문제구분 / Main_문제번호_작성자이름.java
  ```
  gildong/week3/common/Main_1000_송해찬.java
  gildong/week2/personal/Main_2000_송해찬.java
  ```
  * 문제구분
    + common : 공통 문제
    + personal : 개인 문제
